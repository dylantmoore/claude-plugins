# Dylan Moore's Claude Code Plugins

A marketplace of Claude Code plugins. Add it once, then install any plugin listed below.

## Setup

```
/plugin marketplace add dylantmoore/claude-plugins
```

## Available Plugins

### Stata

| Plugin | Description | Install |
|--------|-------------|---------|
| [stata](https://github.com/dylantmoore/stata-skill) | Comprehensive Stata reference — syntax, data management, econometrics, causal inference, graphics, Mata, and 20 community packages. | `/plugin install stata` |
| [stata-c-plugins](https://github.com/dylantmoore/stata-skill) | C/C++ plugin development for Stata — SDK setup, .ado wrappers, cross-platform compilation, performance optimization, and Python/R package translation. | `/plugin install stata-c-plugins` |
| [stata-bundle](https://github.com/dylantmoore/stata-skill) | Both Stata skills in one install. | `/plugin install stata-bundle` |

### Overleaf

| Plugin | Description | Install |
|--------|-------------|---------|
| [overleaf-cli](https://github.com/dylantmoore/overleaf-cli) | Interact with Overleaf LaTeX projects — read, edit, compile, search, manage files, download PDFs, and more. | `/plugin install overleaf-cli` |
